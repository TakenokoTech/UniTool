UniTool

Unityの便利関数を集めたライブラリです。

Quick Start

  1. GitHubのリポジトリで最新のunitypackageをダウンロード
  2. Unityプロジェクトに取り込み

Unity PackageManager

  1. Packages/manifest.jsonに以下を追加。
{
  "dependencies": {
    ...
+   "tech.takenoko.unitool": "https://github.com/TakenokoTech/UniTool.git?path=/Assets"
  }
}
